**GG eBay Management** helps you manage eBay accounts, payment accounts, server accounts,...

## Features
* Create and manage eBay accounts
* Create and manage payment accounts
* Create and manage server accounts
* Filter and search accounts
* Search by account meta: SKU, email, user name, ...

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended
* WooCommerce version 3.6.
